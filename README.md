## 项目概述

狼人杀Wiki是一个开源的知识库项目，致力于记录和分享狼人杀游戏的所有相关知识。通过清晰的结构、详细的说明和实用的攻略，帮助玩家更好地理解游戏机制、掌握游戏技巧，提升游戏体验。

## 快速开始

### 环境准备

在开始之前，您需要安装Python和MkDocs。以下是安装步骤：

1. 安装Python（推荐Python 3.8或更高版本）
2. 安装MkDocs和Material主题：

```bash
# 安装MkDocs
pip install mkdocs

# 安装Material主题
pip install mkdocs-material

# 安装额外插件（可选）
pip install mkdocs-git-revision-date-plugin mkdocs-awesome-pages-plugin mkdocs-material-extensions
```

### 本地开发

克隆仓库后，在项目根目录运行以下命令启动本地服务器：

```bash
mkdocs serve
```

然后访问 http://localhost:8000 查看网站。本地服务器支持热重载，修改文件后会自动更新网站内容。

### 构建网站

完成编辑后，可以使用以下命令构建静态网站：

```bash
mkdocs build
```

构建后的文件将生成在`site`目录中，可以部署到任何静态网站托管服务（如GitHub Pages、Netlify、Vercel等）。

## 如何编辑Wiki

狼人杀Wiki使用Markdown语法编写文档，以下是详细的编辑指南：

### 文档结构

项目的文档内容位于`docs`目录下，主要包括以下文件和目录：

- `index.md`：网站首页
- `about.md`：关于页面
- `roles/`：角色介绍目录，包含各阵营角色文档
- `rules.md`：游戏规则
- `judge.md`：法官话术
- `strategy.md`：游戏攻略
- `faq.md`：常见问题
- `event.md`：赛事信息

### 编辑文件

1. 找到您想要编辑的文件，使用任何文本编辑器（如VS Code、Sublime Text等）打开
2. 使用Markdown语法进行编辑
3. 保存文件后，使用`mkdocs serve`命令启动本地服务器，查看修改效果
4. 确认无误后，可以提交修改

### Markdown基础语法

以下是常用的Markdown语法：

- 标题：使用`#`符号，例如`# 一级标题`，`## 二级标题`
- 加粗：使用`**`包裹，例如`**加粗文本**`
- 斜体：使用`*`包裹，例如`*斜体文本*`
- 列表：使用`-`或数字加`.`，例如`- 列表项`或`1. 第一项`
- 链接：`[链接文本](链接地址)`，例如`[百度百科](https://baike.baidu.com/)`
- 图片：`![图片描述](图片路径)`
- 代码块：使用三个反引号包裹，并指定语言，例如
````
```python
print("Hello World")
```
````

- 表格：
```
| 列1 | 列2 |
| --- | --- |
| 内容1 | 内容2 |
```

### 导航配置

网站的导航结构在`mkdocs.yml`文件中配置，您可以修改此文件来调整网站的导航结构。导航配置示例：

```yaml
nav:
  - 首页: index.md
  - 角色介绍:
      - 角色概述: roles/overview.md
      - 好人阵营: roles/good.md
      - 狼人阵营: roles/werewolf.md
      - 第三方阵营: roles/third.md
  - 游戏规则: rules.md
  - 游戏术语: roles/terms.md
```

### 添加新页面

要添加新页面，请按照以下步骤操作：

1. 在`docs`目录下创建新的Markdown文件，例如`new_page.md`
2. 在`mkdocs.yml`文件的`nav`配置中添加新页面的导航条目
3. 编辑新页面的内容
4. 使用`mkdocs serve`命令查看效果
